### Added
- Project scaffolding: Vite + React 18 + Tailwind CSS v3
- Design tokens: color palette, typography scale, spacing in `tailwind.config.js`
- Base UI primitives: `Button`, `Container`, `SectionHeader`, `Reveal`
- Motion helpers: `SplitText` for hero headline, `Reveal` wrapper for scroll-triggered entrance
- `Nav` — sticky with `backdrop-blur` on scroll, mobile hamburger menu
- `Footer` — four-column layout with decorative links and copyright
- `Hero` — split layout, animated headline, phone mockup entrance
- `Stats` — animated counters with stagger
- `Features` — 6-card grid with stagger reveal
- `Showcase` — alternating mockup blocks with parallax scroll
- `Testimonials` — 3-quote grid, horizontal scroll snap on mobile
- `Pricing` — Free / Pro / Team cards, monthly/yearly toggle
- `FAQ` — animated accordion via Framer Motion `AnimatePresence`
- `FinalCTA` — full-width dark section with decorative watermark
- Static content data files: `features.js`, `stats.js`, `testimonials.js`, `pricing.js`, `faq.js`
- `favicon.svg` — monogram K mark
